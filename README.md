# FIX ME

FIX ME is a smart mobile application that connects customers with skilled workers and contractors through a secure and transparent platform.

## Project Idea

The application aims to improve the labor market experience by helping customers find qualified service providers, communicate with them, send service requests, and evaluate completed work.

## Main Features

- Customer, worker, contractor, and admin roles
- User registration and login
- Service provider browsing
- Service request management
- Chat communication
- Ratings and reviews
- Admin account management
- PHP backend support

## Technologies Used

- Flutter
- Dart
- PHP
- MySQL
- XAMPP
- Firebase
- GitHub
